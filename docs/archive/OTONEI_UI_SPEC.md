# OTONEI 项目 UI 设计规范 (Notion 风格)

本文件详细记录了 OTONEI 音乐项目的 UI 设计规范。本项目视觉风格深度参考 Notion 的极简设计语言，强调信息层级、留白以及优雅的交互反馈。

## 1. 核心颜色层级 (Color Hierarchy)

遵循 L1-L5 逻辑，确保信息密度与视觉重心的平衡。

| 层级 | 颜色代码 | 逻辑定义 | 使用场景 |
| :--- | :--- | :--- | :--- |
| **L1** | `#1F1F1F` | **强调 / 当前选中** | 页面标题、选中项文字、核心按钮文字 |
| **L2** | `#2F3437` | **主要内容** | 正文文本、次级标题、标准按钮文字 |
| **L3** | `#6B6F76` | **辅助信息** | 辅助说明、未选中 Tab 文字、图标颜色 |
| **L4** | `#9B9A97` | **弱化 / 占位** | 搜索框占位符 (Placeholder)、禁用状态 |
| **L5** | `#D3D1CB` | **分割 / 弱化边框** | 边框分割线、进度条背景 |

---

## 2. 背景与边框规范 (Background & Borders)

- **Background (背景)**:
    - 页面背景 (App Background): `#FAFAF9` (Notion Page Background)
    - 容器背景 (Card/Table/Input): `#FFFFFF` (Pure White)
    - 悬停背景 (Hover Background): `#F5F4F2` (Notion Hover)
- **Border (边框)**:
    - 默认边框: `#E9E9E9` (Notion Divider)
    - 激活/焦点边框: `#2F3437`

---

## 3. 按钮系统规范 (Notion-ized Buttons)

为了极致还原 Notion 风格，建议统一按钮交互如下：

### 3.1 交互原则
- **去掉厚重的阴影**：Notion 几乎不使用外阴影，更多依靠 `1px` 细边框或背景色块。
- **轻量化悬停**：悬停时仅改变背景色（通常为 `L5` 变体或淡灰色）或轻微加深文字。

### 3.2 风格建议
- **标准按钮 (Standard)**: 
    - 初始：透明背景或极淡灰色背景，`1px` 边框 (`#E9E9E9`)。
    - 悬停：背景色变为 `#F5F4F2`，边框颜色加深。
- **核心动作按钮 (Primary)**: 
    - 初始：`#2F3437` (L2) 背景，白色文字。
    - 悬停：背景色加深至 `#1F1F1F` (L1)。
- **危险动作按钮 (Danger)**:
    - 初始：红色文字，无背景。
    - 悬停：背景变为淡红色 (rgba(231, 76, 60, 0.05))。

---

## 4. 字体规范 (Typography)

### 4.1 字体族
- **中英文本**: 默认使用 `'Montserrat'`，回退至 `-apple-system`, `BlinkMacSystemFont`, `'Microsoft YaHei'`。
- **数字**: 建议使用等宽字体以确保列表对齐。

### 4.2 级别定义
| 层级 | 字号 (Size) | 字重 (Weight) | 颜色层级 |
| :--- | :--- | :--- | :--- |
| **页面大标题** | `24px` | `700 (Bold)` | `L1` |
| **次级标题** | `16px` | `600 (Semi-Bold)` | `L1` |
| **正文文本** | `14px` | `400 (Regular)` | `L2` |
| **辅助文本** | `12px` | `400 (Regular)` | `L3` |

---

## 5. 圆角与间距 (Radius & Spacing)

### 5.1 圆角规范
- **小组件 (Tag/Badge)**: `4px`
- **交互组件 (Button/Input)**: `6px`
- **容器面板 (Card/Modal)**: `10px` 或 `12px`

### 5.2 布局高度
- **导航栏**: `60px`
- **标准行高**: `35px` - `42px`
